PRIMEIROS PASSOS COM O MKLV

Este é o seu guia!

Bem-vindo ao MKLV, um programa de monitoramento de código open-source. Nossa equipe tem como responsabilidade a criação de um programa que monitora todos os equipamentos computacionais registrados no nome da empresa.

O MKLV é um programa diferente. O MKLV é um site de monitoramento quase que único, sendo que, faz uso de diversas linguagens de programação para interligar Html com Shell Script (bash) também chamado de CGI.

Objetivo

Nosso principal objetivo é monitorar e manter ativo todos os equipamentos, e em caso de problemas resolver o mais rápido possível.

Durante os testes do programa nos foi apresentado uma situação problema, onde um alarme começou a soar, notificando que um equipamento havia parado de responder. Então chegamos a conclusão de que poderia haver dois possíveis problemas:

1º: A possibilidade de que o cabo de rede esteja desconectado ou haja algum problema na rede.

2°: Há também a possibilidade de ser que há algum problema com o hardware.

Se por acaso não for nenhuma das duas opções mencionadas, nos informe onde posteriormente iremos encaminhar o problema para o Departamento de Teste para que eles façam avaliações mais profundas.
